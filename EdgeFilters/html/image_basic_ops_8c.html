It returns the value of the convolution of the matrix in kernel with the equally sized subregion of the image pgmIn1 centered at the linarly indexed pixel ic.

SScan an image and apply a function func to each pixel.

It scans the image pgmIn1 and to each pixel in the image applies the function func. The function func receives 6 parameters:

  • pgmIn1
  • pgmIn2
  • filter->kernel (if != NULL)
  • dimX/2
  • dimY/2
  • a linear index in the pgmIn1->pixels array Each pixel in pgmOut is replaced with the return value of func for the corresponding pixel in pgmIn1.
    Parameters
    pgmIn1Pointer to the first Pgm image structure.
    pgmIn2Pointer to a second Pgm image structure that can optionally be accessed by func.
    pgmOutPointer to the output Pgm image structure.
    filterPointer to an optional Filter structure.
    dimXThe dimension along X of the subarray of pgmIn1 checked by func.
    dimYThe dimension along Y of the subarray of pgmIn1 checked by func.
    funcThe function used to compute the value of the output pixel.
    Returns
    0 on success, -1 if either pgmIn or pgmOut are NULL.

Binarize an image pgmIn based on the threshold value. The result is stored in pgmOut.

Each pixel's value greater or equal than the threshold is converted to the value 255. All other pixels are set to 0.
