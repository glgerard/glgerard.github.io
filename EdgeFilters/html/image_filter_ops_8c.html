execImageOps ( Pgm *  pgmIn,
Pgm *  pgmOut,
FILE *  fp 
)

Filter the image pgmIn with the filters listed in file fp.

The file fp contains a list of filters, one per line, that will be applied in sequence to the image pointed by pgmIn. The output will be stored in the Pgm structure pointed by pgmOut. Certain filters have optional parameters.

Parameters
pgmInPointer to the input PGM image structure.
pgmOutPointer to the output PGM image structure.
fpPointer to a file with the list of filters.
List of implemented filters
  • threshold [threshold_value (default 0)]
  • uniform_noise [range_value (default 32)]
  • salt_n_pepper [density (default 0.05)]
  • normalize
  • equalize
  • median
  • average
  • internal_contour
  • operator_39
  • nagao
  • sharpening
  • prewitt [mod|phase (default mod)]
  • sobel [mod|phase (default mod)]
  • gauss [sigma (default 1)] [dim (default 0)]
  • dog [sigma (default 1)] [dim (default 0)]
  • ced [sigma (default sqrt(2))] [threshold (default 25)]
