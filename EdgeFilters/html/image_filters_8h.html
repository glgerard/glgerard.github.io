Filter* DoGFilter ( double  sigma,
int  dim 
)

Creates a DoG filter with dim columns and dim rows.

This filter is the result of the difference of two equal Gaussian filters with equal size and different sigmas. The external gaussian has sigma set by sigma. The internal gaussian has a smaller sigma equal to sigma / 1.66 .

Parameters
sigmaThe sigma of the external Gaussian.
dimThe number of rows and columns of the filter. If it is set to 0 then the dimension is the smallest odd number next to 6 sigma.
Returns
Pointer to the DoG filter.
